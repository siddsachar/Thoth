Meet Thoth

Your private AI assistant. 19 integrated tools, long-term memory, voice control, vision, inline charts — all running locally on your machine.

⬇ Download for Windows ☆ View on GitHub

Windows 10/11 · 64-bit · Free & open source (MIT)

Everything you need, nothing you don't

🤖

Autonomous ReAct Agent

The agent reasons about your question, decides which tools to use, chains multiple lookups, and assembles a complete answer. Smart context management proportionally shrinks oversized tool outputs so multi-file workflows fit. Live token counter shows context usage in real time.

💬

Persistent Conversations

Chat threads with full history, auto-naming, seamless resume, and export to Markdown, text, or PDF. Attach images, PDFs, CSV, Excel, JSON, and text files directly. First-run welcome guide helps you get started.

🧠

Long-Term Memory

Thoth remembers what you tell it — names, birthdays, preferences, projects. Semantic FAISS search with auto-recall injects relevant memories before every response. Background extraction periodically scans past conversations for personal facts.

👁️

Vision & Screen Capture

Capture images from your webcam or take screenshots and ask questions about them. Uses a local vision model for analysis.

🎤

Voice Input & TTS

Toggle-based voice input with local speech-to-text via Whisper (CPU-only, int8), and neural text-to-speech via Piper. Mic auto-mutes during playback to prevent echo. Fully hands-free — no cloud APIs.

⚡

Workflows

Create reusable multi-step prompt sequences that chain research, summarisation, and action. Run manually or on a daily/weekly schedule. Template variables, background execution, desktop notifications, and 4 built-in templates.

🤖

30+ LLM Models

Switch between Llama, Qwen, Gemma, Mistral, DeepSeek, Phi, and more. Models download automatically on demand via Ollama.

📧

Gmail & Calendar

Search emails, read threads, draft and send messages. View, create, and manage calendar events — all through natural language.

📁

File Management

Read, write, copy, move, and search files within a sandboxed workspace. Reads PDFs, CSV, Excel, and JSON files with schema & stats preview. Interactive Plotly charts rendered inline. Destructive actions require confirmation.

🔒

Fully Local & Private

All LLM inference runs on your machine. No cloud. No telemetry. External APIs are opt-in and individually toggleable.

Up and running in minutes

1

Download & install

Run the installer — it sets up Ollama, Python, and all dependencies automatically. Or clone the repo and run from source.

2

Launch Thoth

Open from the Start Menu, Desktop, or system tray. The app opens in your browser. The default brain model downloads automatically on first launch.

3

Ask anything

The agent will figure out which tools to use. Search the web, check your email, analyze your screen, set a timer — just ask in natural language.

🔒 Privacy by design

Thoth runs entirely on your machine. Your documents, conversations, memories, and API keys are stored locally and never sent to any cloud service. External calls are only made to the services you explicitly enable.
